# 🎓 AI Smart Attendance Management System

An intelligent web-based attendance management system that automates student attendance using Face Recognition technology. The application minimizes manual attendance, improves accuracy, and provides an easy-to-use interface for administrators and students.

---

## 👩‍💻 Developed By

**Sania Singh**

---

## 📖 Project Overview

The AI Smart Attendance Management System is designed to simplify attendance tracking in educational institutions. It captures student facial data, identifies students through a webcam, and automatically records attendance in a MySQL database.

The system includes secure login modules, student management, attendance reports, and profile management.

## ✨ Features

- Secure Admin Login

## 📦 Python Libraries

Install dependencies using:

```bash
pip install -r requirements.txt
```

Common libraries:

- Flask
- OpenCV
- face_recognition
- numpy
- mysql-connector-python
- Werkzeug

---

## 🗄️ Database

Create a MySQL database and import the SQL file.

Example:

```sql
CREATE DATABASE attendance_system;
```

Update your database credentials in `database.py`.

---

## ▶️ Run the Project

Start the Flask server:

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
